## Formulaire de contact

Veuillez utiliser le formulaire disponible à l'adresse https://{{DOMAIN}}/contact avec les informations suivantes :

- **Nom**
- **Adresse e-mail** (afin que nous puissions vous répondre)
- **Message**
- Confirmation obligatoire : « J'ai pris connaissance de la [Politique de confidentialité](/confidentialite). »

Pour savoir comment nous traitons les données transmises via le formulaire, veuillez consulter notre [Politique de confidentialité](/confidentialite). Les informations sur le fournisseur figurent dans les [Mentions légales](/mentions-legales).

Vous pouvez également signaler de tels contenus de manière informelle par e-mail à {{EMAIL}}.

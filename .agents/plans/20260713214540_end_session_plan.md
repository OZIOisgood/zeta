# End Session Plan

## Context

Ticket #34 needs an expert-controlled way to end a live coaching session
without waiting for Agora's idle recording timeout after participants leave.

## Decision

- Only the booking's expert can end it.
- Show the control only while the scheduled session is in progress.
- Persist the early-end state so later connect attempts are rejected.
- Stop an active cloud recording immediately, then return the expert to Sessions.
- Keep cancellation separate: an ended session is completed, not cancelled.

## Scope

- Add a reversible booking end-state migration and sqlc query.
- Add an expert-only end endpoint with recording shutdown and connect enforcement.
- Add the in-call confirmation control and API client/store support.
- Cover time, ownership, connection rejection, and UI behavior with focused tests.

## Verification

Run sqlc generation, focused Go and Angular tests, API/frontend builds, and
record final outcomes in a completion report.

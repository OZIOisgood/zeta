# Upload Page Slides Transition

## Description

Enhance the Video Upload Page UX by replacing the existing stepper or list implementation with a slides-based layout.
Utilize the Taiga UI Slides component (`TuiCarousel` or similar) to manage the transition between upload steps.

## Requirements

-   Integrate `TuiCarousel` (or equivalent Taiga UI component) for step navigation.
-   Improve the overall user experience of the upload flow with smooth transitions.

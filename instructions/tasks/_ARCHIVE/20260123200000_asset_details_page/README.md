# Asset Details Page

## Description

Upon clicking an asset tile on the home screen, the user should be redirected to the Asset Details page (e.g., `http://localhost:4200/asset/{id}`).
The page must display the asset's title and include a video player for playback.

## Requirements

-   Display the asset title.
-   Integrate the Mux Player for video playback.
-   Ensure playback functionality works correctly.
